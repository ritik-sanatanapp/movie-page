# Movie Site
A simple and responsive movie website to view movie details, trailers, and screenshots. Built using Python (Flask) with HTML/CSS frontend. Easy to run and update.

## Features
- View movie details
- Watch trailers
- Browse screenshots
- Responsive (mobile & desktop)
- Supports English & Hindi

## How to Run
1. Clone the repository:
\`\`\`bash
git clone https://github.com/ritik-sanatanapp/movie-site.git
\`\`\`
2. Install dependencies:
\`\`\`bash
pip install -r requirements.txt
\`\`\`
3. Start the application:
\`\`\`bash
python app.py
\`\`\`
4. Open in browser:
http://localhost:5050

## How to Add a New Movie
1. Add movie images to the \`movies/\` folder
2. Update files inside \`templates/\`
3. Save and refresh the browser

## Notes
- Make sure Python is installed
- Recommended: use virtual environment
